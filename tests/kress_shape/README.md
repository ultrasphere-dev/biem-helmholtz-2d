# Known values for $-L_1/2, -L_2/2, M_1/2, M_2/2$ evaluated at $t, \tau \in {0, \frac{2\pi}{5}, ...}$

- `dlp_log.csv` contains $-\frac{L_1 (t, \tau)}{2}$
- `dlp_cont.csv` contains $-\frac{L_2 (t, \tau)}{2}$
- `slp_log.csv` contains $\frac{M_1 (t, \tau)}{2}$
- `slp_cont.csv` contains $\frac{M_2 (t, \tau)}{2}$
- `t`: corresponds to the first axis
- `tau`: corresponds to the second axis
- Contains jacobian for `tau`
